(2)Working with Git Branches

mkdir Git_Exp2
cd Git_Exp2
git init
echo "this is initial content" > file1.txt
git add file1.txt
git commit -m "Initial commit"
git branch feature-branch
git checkout feature-branch
echo "this is feature branch modification" > file1.txt
git add file1.txt
git commit -m "Added new feature in feature branch"
git checkout master
# Modify same file (to force conflict)
echo "this is main branch modification" > file1.txt
git add file1.txt
git commit -m "Updated file in master branch"
# Merge feature branch (conflict will happen)
git merge feature-branch

# -------- CONFLICT RESOLUTION --------
# Open file and fix manually
notepad file1.txt
Inside file, you will see:
<<<<<<< HEAD
this is main branch modification
=======
this is feature branch modification
>>>>>>> feature-branch
Replace with (final resolved content):
this is main branch modification
this is feature branch modification
# Save file, then continue
git add file1.txt
git commit -m "Resolved merge conflict"
git log --oneline
git branch -d feature-branch

(16)Creating Docker Images using Dockerfile

app.py

from http.server import SimpleHTTPRequestHandler, HTTPServer
PORT = 5000
class Handler(SimpleHTTPRequestHandler):
    def do_GET(self):
        self.send_response(200)
        self.end_headers()
        self.wfile.write(b"Hello from Docker")
HTTPServer(("", PORT), Handler).serve_forever()

Dockerfile

FROM python:3.9-slim
WORKDIR /app
COPY app.py .
EXPOSE 5000
CMD ["python", "app.py"]

Run:
docker build -t myapp .
docker run -p 5000:5000 myapp
👉 Open: http://localhost:5000
